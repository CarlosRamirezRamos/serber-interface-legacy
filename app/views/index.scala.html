Servicio de Registro de Bibliotecas y Entidades Relacionadas

Registro estandarizado de datos de bibliotecas y entidades relacionadas, españolas e internacionales, compilado con objeto de dar servicio a la comunidad cultural, al entorno bibliotecario, a la gestión pública y a los ciudadanos.


DEFINICIÓN ISO 2146:2010

Registro: colección de objetos registrados compilada para dar soporte a la actividad de una determinada comunidad.

Participa

Modelo de Datos vanguardista

Datos de registro encapsulados en un modelo de datos versátil y potente que sea recreable, mediante la ejecución de un script SQL DDL, en cualquiera de los sistemas de gestión de bases de datos relacionales disponibles. La base de datos del proyecto piloto funciona sobre H2. Leer más…

Servicios Web RESTful

Aplicación de registro que permite la administración vía web de datos de registro encapsulados en un novedoso modelo de datos. Estos servicios web son RESTful, es decir, funcionan de forma óptima sobre el protocolo HTTP. Leer más…

Interfaz Web responsive

Interacción con registro que permite la gestión vía web de datos de registro encapsulados en un novedoso modelo de datos. Esta interfaz web es responsive, es decir, funciona de forma óptima sobre cualquier dispositivo. Leer más…

Objetivos del proyecto piloto

El proyecto piloto pretende demostrar algunas de las características del proyecto SERBER:

  • Síntesis de un novedoso modelo de datos basado en el esquema de la ISO 2146:2010 que permita encapsular datos de registro procedentes de ALZIRA/DIANA.

  • Construcción de un servicio de registro dotado de un API de servicios web RESTful que permita la obtención (modo lectura) de datos de registro encapsulados en un novedoso modelo de datos. Utilización de software libre y código abierto.

  • Construcción de una aplicación web, "prueba de concepto" del servicio de registro, dotada de una interfaz web responsive de búsqueda que utilice un API de servicios web RESTful para la obtención (modo lectura) de datos de registro encapsulados en un novedoso modelo de datos. Utilización de software libre y código abierto.
