Interfaz Web

Interacción con registro que permite la gestión web de datos de registro desde cualquier dispositivo.

  1. La interacción con registro proporciona un mecanismo de gestión que permite buscar, visualizar y depurar datos de registro mediante una interfaz web.

  2. La interfaz web habilita la posibilidad de manejar orígenes de datos, vocabularios controlados y vigilancia de duplicados.

  3. La interfaz web es responsive, es decir, fluida y adaptable. Una interfaz web responsive permite que la interacción con los contenidos ocurra sobre cualquier dispositivo de forma óptima.
