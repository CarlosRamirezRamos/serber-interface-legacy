  1. Database Schema

    El modelo de datos se basa en el esquema de la ISO 2146:2010 ya probado e implementado por ANDS. Este modelo de datos ha demostrado ser perfecto para encapsular datos de diferentes fuentes (origen) y, por tanto, de diferentes tipos (naturaleza).

  2. Documentación navegable del esquema de la ISO 2146:2010

    El modelo de datos ha sido sintetizado a partir de las estructuras XML que implementan el esquema de la ISO 2146:2010. Existe una extensa documentación acerca de dicho esquema. Resulta muy recomendable seguir de cerca dicha documentación para no perderse en los campos uno-a-varios, los campos complejos, las relaciones de múltiples niveles o los "pares" type/value.

    • Registry Objects Schema
    • Activity Schema
    • Collection Schema
    • Party Schema
    • Service Schema
    • Registry Types Schema
