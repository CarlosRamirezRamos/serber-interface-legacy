@(title: String, menu: String) @serber("Servicios Web", "API de Servicios Web RESTful que permita el intercambio de información con el origen de datos y con terceros.", title) {

  1. Los servicios web admitirán diferentes formatos: JSON y XML.

    • JSON: Especialmente útil resulta para el uso por interfaz web y su consumo desde clientes web, es muy utilizado incluso para bases de datos no relacionales como MongoDB. En este caso se utilizará para intercambiar datos con el modelo de datos.
    • XML: Especialmente útil resulta para el intercambio entre máquinas o aplicaciones. En este caso se utilizará para el esquema de la ISO.

  2. Los servicios web RESTful son ligeros y estándares, además permiten la obtención de datos vía HTTP con un tráfico mínimo.

  3. API de servicios web RESTful:

    • (GET) /api/registry-object
    • (POST) /api/registry-object/:id
    • (GET) /api/registry-object/:id
    • (PUT) /api/registry-object
    • (DELETE) /api/registry-object/:id
    • …
}
